# Roman Calculator

author: siritas s.<siritas@gmail.com>

To run a program

```
java -jar roman-calculator-1.0.20170330.jar -f input.txt
```

OR

```
java -jar roman-calculator-1.0.20170330.jar V+V
```

To build a jar

```
mvn package
```
The `roman-calculator-1.0.20170330.jar` file is under `target` folder.
